Cara kerja

Siap digunakan dalam tiga langkah.

1

Instal aplikasi macOS

Karyawan menginstal BiBoTracking dan memberikan izin Accessibility, Input Monitoring, dan Screen Recording. Status diperbarui otomatis saat izin diberikan.

2

Tambahkan ekstensi Chrome

Opsional. Ekstensi Manifest V3 menemukan aplikasi lokal secara otomatis dan melaporkan halaman browser ke 127.0.0.1 — tidak pernah ke cloud. Dapatkan di Chrome Web Store →

3

Pemilik memantau rangkuman

Aktivitas tersinkronisasi satu arah ke dashboard, tempat pemilik meninjau waktu, ketukan tombol, riwayat browser, dan screenshot per karyawan.

Lihat self-hosting

Open source

Tidak mau bayar? Hosting sendiri.

BiBoTracking bersifat open source — backend Go, dashboard web, aplikasi macOS, dan ekstensi Chrome semuanya milik Anda untuk dibaca, diaudit, dan dijalankan di mesin Anda sendiri, gratis selamanya. Layanan hosted di sini hanyalah pilihan yang praktis.

Self-host seluruh stack

Satu binary Go melayani API dan dashboard, didukung oleh Postgres. Infrastruktur Anda, aturan Anda.

Periksa segalanya

Baca source code-nya dan verifikasi persis apa yang ditangkap — dan apa yang tidak pernah meninggalkan perangkat.

Tanpa lock-in

Data Anda hidup di SQLite dan Postgres Anda sendiri. Ekspor, pindahkan, simpan — kapan saja.
