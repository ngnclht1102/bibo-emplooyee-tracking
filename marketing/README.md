## Brand

**Name:** Employee Tracker ¬∑ **Mark:** a person silhouette inside a clock ring
(employee + time). **Font:** San Francisco / system stack
(`-apple-system, BlinkMacSystemFont, "Segoe UI", system-ui, sans-serif`).

## Suggested copy

- **Tagline:** Local-first time & activity tracking for macOS.
- **Sub:** See where work time goes ‚Äî app usage, keystrokes, screenshots, and
  browser time ‚Äî all stored on the machine, exported on your terms.
- **Feature blurbs:**
  - **App & window time** ‚Äî idle-aware, so only active time counts.
  - **Keyboard activity** ‚Äî counts only; the actual keys are never recorded.
  - **Periodic screenshots** ‚Äî automatic, with retention limits.
  - **Browser pages & time** ‚Äî via a lightweight extension.
  - **100% local** ‚Äî no cloud, no account; export to CSV or JSON anytime.
  - **Menu bar** ‚Äî start/stop and at-a-glance status (green / amber / red).

> Design note: flat, low-color, one accent ‚Äî keep marketing visuals consistent with
> the app's quiet utilitarian style.
