## Brand

**Name:** Employee Tracker · **Mark:** a person silhouette inside a clock ring
(employee + time). **Font:** San Francisco / system stack
(`-apple-system, BlinkMacSystemFont, "Segoe UI", system-ui, sans-serif`).

## Suggested copy

- **Tagline:** Local-first time & activity tracking for macOS.
- **Sub:** See where work time goes — app usage, keystrokes, screenshots, and
  browser time — all stored on the machine, exported on your terms.
- **Feature blurbs:**
  - **App & window time** — idle-aware, so only active time counts.
  - **Keyboard activity** — counts only; the actual keys are never recorded.
  - **Periodic screenshots** — automatic, with retention limits.
  - **Browser pages & time** — via a lightweight extension.
  - **100% local** — no cloud, no account; export to CSV or JSON anytime.
  - **Menu bar** — start/stop and at-a-glance status (green / amber / red).

> Design note: flat, low-color, one accent — keep marketing visuals consistent with
> the app's quiet utilitarian style.
