## Detailed description
BiBo Tracker helps teams understand how work time is spent — without sending anything to the cloud.

This browser extension records the active tab (page URL, title, and time on page) and reports it ONLY to the BiBo Tracker desktop app running on the same computer, over the local loopback address (127.0.0.1). Your data never leaves the device, and the publisher receives nothing.

WHY TEAMS USE IT
• Accurate time-on-task and top-sites breakdowns for the work day
• Local-first: all activity is stored on the employee's own device, inside the desktop app your organization controls
• No cloud account, no analytics, no third-party tracking, no ads
• Lightweight Manifest V3 service worker — minimal overhead

PRIVACY YOU CAN VERIFY
• We collect zero data. Nothing is transmitted to the developer or any external server.
• The extension only talks to localhost (127.0.0.1) — your own machine.
• It does not read page content, form inputs, passwords, cookies, or keystrokes.
• Source code is available for inspection so you can confirm exactly what it does.

CONSENT & RESPONSIBLE USE
BiBo Tracker is intended for workplace use with the knowledge and consent of the people using the device, in line with your organization's policies and local laws.

HOW IT WORKS
1. Install the BiBo Tracker desktop app on the work computer.
2. Add this extension to the browser.
3. The extension automatically pairs with the local app and begins recording time on page — all stored locally.
